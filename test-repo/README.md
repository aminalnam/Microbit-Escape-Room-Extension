# Escape Room Puzzle Projects

Seven ready-to-use MakeCode projects. Each one is a complete,
commented escape room puzzle you can customise and flash.

## How to open in MakeCode

1. Go to **makecode.microbit.org**
2. Home screen → **Import** → **Import URL**
3. Paste the GitHub URL for the folder you want:

```
https://github.com/YOUR_USERNAME/microbit-escape-room-tests/tree/main/puzzle1-sequence
https://github.com/YOUR_USERNAME/microbit-escape-room-tests/tree/main/puzzle2-pin
https://github.com/YOUR_USERNAME/microbit-escape-room-tests/tree/main/puzzle3-morse
https://github.com/YOUR_USERNAME/microbit-escape-room-tests/tree/main/puzzle4-pattern
https://github.com/YOUR_USERNAME/microbit-escape-room-tests/tree/main/puzzle5-light
https://github.com/YOUR_USERNAME/microbit-escape-room-tests/tree/main/puzzle6-compass
https://github.com/YOUR_USERNAME/microbit-escape-room-tests/tree/main/puzzle7-shake
```

Replace YOUR_USERNAME with your GitHub username.

## Puzzles

| # | Puzzle | Secret | Controls |
|---|--------|--------|---------|
| 1 | Button Sequence | A, B, A | A/B to press, A+B for hint |
| 2 | PIN Code | 3 7 2 | A=scroll digit, B=confirm, A+B=submit |
| 3 | Morse Code | SOS | A=dot, B=dash, A+B=confirm letter |
| 4 | LED Pattern | Letter T | A=move cursor, B=toggle, A+B=submit |
| 5 | Light Sensor | Level 50–150 | A=show reading, A+B=submit |
| 6 | Compass | East (90°) | A=show heading, A+B=submit |
| 7 | Shake | Shake! | Shake, A+B for hint |

## Customising a puzzle

Each project has clear comments explaining what to change.
Open any project, go to the JavaScript tab, and edit the
values at the top of the file.
