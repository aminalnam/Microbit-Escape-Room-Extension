# Microbit Escape Room Extension — Examples (Using Extension)

These example projects are block-safe and USE the extension API.

## How to use
1. Open https://makecode.microbit.org
2. Click Import → Import URL
3. Paste a puzzle folder URL (from your GitHub)

## Requirements
- Add your extension via Extensions → paste your GitHub repo URL:
  https://github.com/aminalnam/Microbit-Escape-Room-Extension

## Conventions
- Radio group: 42 (set via extension)
- Messages handled by extension: "door-open", "__RESET__"

## Notes
- Puzzle logic is kept in the project for Blocks compatibility.
- Extension is used for radio/grouping and events.
